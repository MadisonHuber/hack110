Title

COMP110 Review


Description of Project

This project is a review of COMP110. It contains multiple pages reviewing the concepts covered in COMP110 with explanations, descriptions, examples, and additional resources related to each specific topic. It also contains a page that lets you search through an array of people objects based on different criteria.


Team Members

Madison Huber


110 Programming Concepts Used

This project mainly used html, but the search page required use of if/else-if logic, events, using HTMLElements in Typescript, arrays, and for loops. Additionally, each page focused on different COMP110 topics, so making those examples touched on some COMP110 topics as well.


Challenges Overcome / New Concepts Learned

I got more familiar with using html, css, and bootstrap. For example, I learned how to create a navigation bar using bootstrap. At first, I had some challenges getting changes from the Typescript files to affect the corresponding html files, but I eventually got a better grasp of using HTMLElements in Typescript.
