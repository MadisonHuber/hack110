Loops

Overview

Loops make it nicer to do something repeatedly. Instead of rewriting the same code each time we want to use it, we can put it in a loop and tell the loop how many times to do it.

Description

We can use loops to do stuff over and over. To do so, we need to specify how many times the loop should execute and keep track of how many times it has already executed. We need to be sure to give a definite condition that will eventually be false so as to avoid infinite loops.

Making Loops

When using loops, we can either use a while loop or a for loop to get the job done.

While Loops


        while (condition) {
            // do something
        }

When using while loops, we usually need to create a counter variable before entering the loop. This will be incremented within the loop, then used with the given condition. When the condition evaluates to true, we enter the loop, and when it's false we don't.

Example While Loop


        let i: number = 0;
        while (i < 10) {
            print(i);
            i++;
        }

For Loops


        for (let counter: number = value; condition; counter++) {
            // do something
        }

In a for loop, the creation of the counter variable, the condition, and the incrementing of the counter all happen within the parentheses (). First, the counter is created, then the condition is checked, then the stuff within the loop happens, then the counter is incremented, then the condition checked again....

Example For Loop


        for (let i: number = 0; i < 10; i++) {
            print(i);
        }

Depending on the condition, we may need to decrement the counter or make other modifications to it instead of incrementing so that it will get closer to stopping the loop by making the condition false.
