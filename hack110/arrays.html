Arrays and Multidimensional Arrays

Overview

Arrays are lists that will hold things for us. They can hold a bunch of things at once, and they are reference types.

Description

Arrays hold a bunch of elements as in a list. They are indexed beginning at 0. There is a separate element at each index, and you are allowed to have elements repeat.

The elements we put in our array can be whatever we want-value types or reference types.

We can think of arrays in many different ways, so we'll have some fun with this. We'll think of arrays in terms of a grocery store.

Some grocery stores conveniently mark their aisles with signs that number the aisles and describe what each aisle contains. We'll use these to help us understand what an array is and how it works.

How We Use Arrays

The signs marking grocery store aisles are great because they tell us information about what's in the aisle. As we know, types help us with this in our code. Just like everything else, when we make an array we must give it a type. Then, we can start to fill in the array we make. Typescript arrays are nice because we can always add onto them if we want, and there're a lot of other cool things we can do with them, too. Here's how we make one:


        let arrayName: type[] = [];

Just like when we make other variables, we made sure to give a type. This type included the square brackets ([]) which tell us it's an array. We then initialized it in the same line (like we can with other variables). We initialized it to be an empty array.

Examples

The labels on our aisle sign are colas, potato chips, soft drinks, nuts, and snacks. We can handle this! These are pretty fun foods, so we could make a PartyFood class. Great! One thing we know about arrays is that we usually like to keep all our data within the array the same type. So, maybe all these foods fit within the PartyFood class, and our array can hold these foods by being an array of PartyFood objects. Here's how we'd do that:


        class PartyFood {
            item: string;
            numberInStock: number = 0;
            constructor(item: string) {
                this.item = item;
            }
        }

        let aisleEleven: PartyFood[] = [];

With this, we've just created a PartyFood class that has an item property which is initialized in the constructor. We also created a new array (let aisleEleven: PartyFood[]) and initialized it (= []) all in one line. As we know, we could have broken this up into separate lines if we wanted to. Now we just need stuff to put in our array!


        let aCola: PartyFood = new PartyFood("cola");
        let aPotatoChip: PartyFood = new PartyFood("potato chips");
        let aSoftDrink: PartyFood = new PartyFood("soft drinks");
        let aNut: PartyFood = new PartyFood("nuts");
        let aSnack: PartyFood = new PartyFood("snacks");

Now we have a bunch of PartyFood objects. Since our array is an array of PartyFood objects, we can use these objects we just made to fill our array!


        aisleEleven = [aCola, aPotatoChip, aSoftDrink, aSoftDrink];

Uh-oh! It looks like we accidentally put aSoftDrink in there twice instead of putting in aNut! Let's fix this by going in our array and accessing then reassigning what's in the specific index where we have our second aSoftDrink.


        aisleEleven[3] = aNut;

This is how we access a specific index in an array. This format (arrayName[indexNumber] = element;) lets us change the element stored at that index of the array. It's important to remember that arrays are zero-indexed, which is why changing the element at index 3 to be aNut puts in aNut in place of the second aSoftDrink. The modified array is [aCola, aPotatoChip, aSoftDrink, aNut].

Oh no! We also forgot to add aSnack to our array! Turns out, we can just append it on the end of the array to get the job done. Here's how:


        aisleEleven[aisleEleven.length] = aSnack;

Why aisleEleven.length? The array we were adding to had 4 elements in it (aCola, aPotatoChip, aSoftDrink, aNut). We also know arrays are zero-indexed, so aCola had an index of 0, aPotatoChip 1, aSoftDrink 2, and aNut 3. By saying aisleEleven[aisleEleven.length], this takes whatever the length of the array was (in this case 4) and makes that the index we are accessing. So this added the next index in which we put our aSnack element.

So far, we've created an array holding objects, modified its contents, and added onto it, but there's still more we can do with arrays!

Iterating through Arrays

Now we've got a nice aisle of party food, except that each item still has a numberInStock value of 0! We need to go through and stock the shelf so that there are three of each item. We'll want to access each element of the array and change its numberInStock property to 3 instead of 0. We can use a loop (yay!) to iterate through the array to have access to each element. To increase the numberInStock value from 0 to 3, we can either directly assign it the value of 3 or use another loop to increment one by one until we reach 3. Since we'd have to stock items one at a time, we'll use a loop to get to three.


        for (let i: number = 0; i < aisleEleven.length; i++) {
            for (let count: number = aisleEleven[i].numberInStock; count <= 3; count++) {
                aisleEleven[i].numberInStock = count;
            }
        }

We just used nested for loops to iterate through our array and reassign each element's numberInStock property. We now know how to make an array, fill it, access and edit an element at a specific index, add an element to the array, and go through its elements one by one.

2D Arrays

We can pretty much have arrays of any type, and we know we must specify this type upon creation of the array. This means we can have arrays of arrays! We can make a 2D array the same way as we made our previous array, but the type of our 2D array is now an array as well. Let's check out how we'd do this:


        let nameOfArray: type[][] = [];

We've got this part down by now. Here, we haven't really done anything new, except that the type is now an array.

Examples with 2D Arrays

We'll go back to our grocery store with all the party food. Apparently, people love to party because our grocery store has decided it would like to add a special display for the PartyFood drinks, which will include cola and soft drinks. We know how to do this:


        let display: PartyFood[] = [aCola, aSoftDrink];
        for (let n: number = 0; n < display.length; n++) {
            for (let count: number = display[n].numberInStock; count <= 6; count++) {
                display[n].numberInStock = count;
            }
        }

This is pretty much what we did before. We made an array then went through it to change the numberInStock to 6, since this is a big display. Great! Now we want to be able to put all our party foods together in one group to stay organized. Here's what we'll do:


        let allOurPartyFood: PartyFood[][] = [aisleEleven, display];

Perfect! Now we've got an array of our PartyFood arrays. What now? Well, there's a big sale, and the store sells out of everything, so we need to update our numberInStock property for each object. We'll do so like this:


        for (let row: number = 0; row < allOurPartyFood.length; row++) {
            for (let col: number = 0; col < allOurPartyFood[row].length; col++) {
                while (allOurPartyFood[row][col].numberInStock > 0) {
                    allOurPartyFood[row][col].numberInStock--;
                }
            }
        }

Okay! We made our 2D array and we've worked our way through it using loops. Now that 2D arrays (and arrays in general) are making sense, we'll know how to use them. Additionally, we know that if we wanted to, we could apply similar steps to make arrays with more than two dimensions.
