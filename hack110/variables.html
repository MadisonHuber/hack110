Variables and Value Types

Overview

Variables are like tupperware containers-they are used to hold things so you can come back for them and use them later.

Types help us know more about our variables so we know what we're doing. When something is given a type, it is put into that category, which is useful becasue different things can be done with different types.

Description

Variables can hold different things which fall into different categories:

  • number
    • Numeric values
    • Examples: 0, 1, 4535, -34, 0.45
  • string
    • Words, or individual characters strung together
    • Examples: "the", "75", ":)"
    • strings are enclosed in "", and anything enclosed in "" is generally considered a string
  • boolean
    • Values that are either true or false
    • Examples: true, false

These categories are value types. The value your variable holds fits into one of these categories.

Great! Now back to tupperware and food...
So, variables are like tupperware. If I'm really hungry and order a pizza, chances are I'm going to eat 7 out of the 8 slices. I have an extra piece, but there's no way I'm throwing it away! I'll store it in my tupperware, and next time I want to do something with that piece, such as eat it or reheat it, I'll grab my tupperware to get my pizza!
What if I only end up eating 6 pieces? I could get a separate container for each slice, but maybe it'd be nice to keep them together so they don't get lonely in my fridge. I can easily just put them both in one tupperware.

Guess what? Variables can hold more than one value, but things get a little more complicated with this. Now, the variable isn't just storing one value, just as the tupperware isn't just storing one piece of pizza. When a variable holds anything other than one simple value (one string, one number, or one boolean), it becomes a reference type!

Okay, but what if I don't feel like pizza and instead get cake? I love pizza and I love cake, but I generally don't like mixing my pizza and cake together. If I wanted to store leftover cake, I probably won't put it together with my pizza all in one tupperware.

Similarly, we generally want whatever is held by our variable to all be the same type.

Making and Using Variables

There are different ways to make a variable, but we generally follow this format:


            let variableName: type;
            variableName = value;

In the first line, we create a new variable, and in the second line we give it a value. This can be done in one line:


            let variableName: type = value;

Example:


            let name: string = "Kris";

We generally use 'let' when we create a variable, but there are other ways to do so. For example, 'const' creates a constant variable whose value cannot be changed. Variables created with 'let' can be reassigned to new values, as done below with the name variable we just created. When we want to access or reassign the variable's value, all we have to do now is use the name of our variable.


            name = "Jeffrey";
