Boolean Expressions

Overview

We already know booleans are just true or false. The fun part is that we can have expressions that evaluate to true or false and are thereby booleans.

Description

As long as it evaluates to true or false anything can be a boolean. We can do this by using any of the relational operators to make comparisons:

  • >
    • Greater than
  • >=
    • Greater than or equal to / at least
  • <
    • Less than
  • <=
    • Less than or equal to / at most
  • ===
    • Equal to
  • !==
    • Not equal to

Relational operators are used to make conditional statements that will evaluate to true or false.

Examples


            let test: boolean = 7 > 5; // evaluates to true
            let num: number = 4;
            let newTest: boolean = num <= 5; // evaluates to true

Logical Operators and Negation

We can make even more complex booleans by using logical operators and negation:

  • &&
    • and
    • if b1 and b2 are variables that are of type boolean, we can use the and operator by saying: b1 && b2
    • evaluates to true only if each thing being compared (b1 and b2) evaluates to true
  • ||
    • or
    • if b1 and b2 are variables that are of type boolean, we can use the or operator by saying: b1 || b2
    • evaluates to true if one or both of the things being compared (b1 and b2) evaluate to true
  • !
    • negation / not operator
    • ! is used to take the flipped meaning of something. It can be used on any statement to flip its meaning.
    • For some boolean that normally evaluates to true, adding the negation operator now gives us false
    • let newBoolean: boolean = 4 < 6; // evaluates to true
      let tester: boolean = !newBoolean; // evaluates to false
    • !(3 > 9) evaluates to true

These are often used to create conditional statements.

Conditional Statements, if, if-else, else-if

We can use our knowledge of booleans and the relational, logical, and negation operators to create conditional statements. These often take the form:

  • if (condition) {
    // do something
}

  • if (condition) {
    // do something
} else {
    // do something different
}

  • if (condition) {
    // do something
} else if (different condition) {
    // do something different
} else {
    // do some other thing
}

If we make an if statement, we can add on as many else-ifs as we want. There are also many ways we can nest these. For example, we could put an if-else block as the "do something" part of an outer if block.

Note on Using Return with Conditionals

Within our if, if-else, or else-if blocks, we can do a lot of different things. One thing we can do if we are within a function is return. We can put a return statement in each block, but only one will be executed, and once one is executed, the execution of the function stops.
